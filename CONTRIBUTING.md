# Contributing

Contributions to this profile repository are welcome.

## How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-change`
3. Make focused, clear changes
4. Commit with a concise message
5. Open a pull request

## Guidelines

- Keep the README professional and accurate
- Do not commit secrets, tokens or private credentials
- Prefer small, reviewable changes

## Issues

If you find a problem, open an issue with:

- What is wrong
- Steps to reproduce
- Suggested fix (optional)
